Key Features ==========

- Add support for custom implementations of _PSU

        implementations
    - Make a factory function for taking PSUConfig objects and turning them into
        intialized instances of the right class
    - Pass IP/Serial config data from PSUConfig to the implementation
    - We will include implementations for Anagate CAN and CAEN HV
- Add new Event types
    - PSU settings change
    - Run arbitrary script (Tamalero)
    - DAQ waveform retrieval
    - Interlock trigger (+ DAQ waveform triggered by falling edge)
- Add PSU dependencies (e.g. PSU X must be enabled before PSU Y)
    - Config stored in EnvironmentConfig
    - Handled by _PSU in EnvironmentControl
    - Best to measure actual output voltage rather than trust setting
- Add Tamalero tests
    - Settings stored in TestSettings (under Criterion)
    - Handled by new class in EnvironmentControl
    - Will require calling existing Tamalero scripts
    - Monitor/measure output and eval PASS/FAIL?

    - Config stored in EnvironmentConfig (under new classes)
    - Settings stored in TestSettings (under Criterion)

        unlimited variety of methods for toggling a physical pin
